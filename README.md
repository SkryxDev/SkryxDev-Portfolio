# 🌐 Portfolio - skryxdev.eu

Questo repository contiene il mio portfolio personale realizzato con
**Vite**, **React** e **TypeScript**.\
Il sito è disponibile online su: https://skryxdev.eu

------------------------------------------------------------------------

## 🚀 Tecnologie utilizzate

-   ⚡ Vite
-   ⚛️ React
-   🔷 TypeScript
-   🎨 CSS
-   📦 npm

------------------------------------------------------------------------

## ▶️ Avvio in locale

Clona il repository:

``` bash
git clone https://github.com/skryxdev/SkryxDev-Portfolio.git
cd portfolio
```

Installa le dipendenze:

``` bash
npm install
```

Avvia il server di sviluppo:

``` bash
npm run dev
```

------------------------------------------------------------------------

## 🏗️ Build per produzione

Per generare la build ottimizzata:

``` bash
npm run build
```

Anteprima della build:

``` bash
npm run preview
```

------------------------------------------------------------------------

## 🌍 Deployment

Il progetto è hostato su un server self-hosted con:

-   NGINX come web server
-   Nginx Proxy Manager per gestione dominio e SSL
-   Let's Encrypt per certificato HTTPS

Dominio:

    https://skryxdev.eu
